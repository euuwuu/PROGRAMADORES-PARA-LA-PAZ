Cuando se envía un reporte mediante la ruta POST /reportes, el servidor guarda la información en el array llamado reportes. Al consultar la ruta GET /reportes, se muestra una lista con todos los reportes registrados mientras el servidor permanece encendido.

Ejemplo de respuesta:

[
  {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Daño en alumbrado público"
  }
]
